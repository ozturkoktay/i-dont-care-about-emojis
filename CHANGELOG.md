# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-28

### Added
- Initial release of I don't care about emojis extension
- Smart emoji detection using Unicode ranges
- Dual blocking modes: Desaturate (black & white) and Hide
- Whitelist functionality for allowing emojis on specific websites
- Real-time processing of dynamic content using MutationObserver
- Cross-browser support for Chrome and Firefox
- Popup UI for quick settings access
- Options page for advanced configuration
- Background service worker for cross-page communication
- Clean architecture following SOLID principles
- Comprehensive test suite
- CI/CD pipeline for automated publishing
- ESLint and Prettier for code quality
- Complete documentation

### Technical Features
- Manifest V3 support
- ES6+ modules
- CSS filters for emoji desaturation
- Chrome Storage API for settings persistence
- Efficient DOM processing without affecting images

### Documentation
- README with detailed usage instructions
- CONTRIBUTING guide for developers
- Architecture documentation
- CI/CD setup instructions
- Code examples and API documentation
